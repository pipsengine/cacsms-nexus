# Cacsms EA (System Experts Directory)

Canonical Expert Advisor source folder for Cacsms Nexus.

- Deploy compiled `.ex5` and source `.mq5` builds here.
- Link this folder to each MT5 terminal `MQL5/Experts` path from the **EA & Terminal Hub** console.

Default path: `C:\Next-Generation\cacsms-nexus\services\cacsms-ea`

Override with environment variable `CACSMS_EA_ROOT` in `apps/web/.env.local`.
